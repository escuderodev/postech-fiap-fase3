# == anotações importantes ==

- criar projeto com: npm create vite@latest list-app --template react-ts
- digitar nop terminal npm install para instalar todas as dependências
- dentro de src, criar pasta components
- dentro de componets devemos criar pasta AddTask
- dentro da pasta AddTask, criar arquivo AddTask.tsx
- dentro de componets devemos criar pasta ListTask
- dentro da pasta ListTask, criar arquivo ListTask.tsx
- no arquivo App.tsx, importar os arquivos AddTask.tsx e ListTask.tsx
